You are running a standalone UX Designer review.

## What This Does

Activates only the UX Designer agent from the product quartet. Use this when you want a focused UX perspective without running the full quartet.

## Process

Use the product-quartet:ux-designer skill. The UX Designer will:

1. Ask clarifying questions about the user, their context, and the feature
2. Define the core flow step by step
3. Identify friction points and edge cases
4. Produce a complete UX Brief

The UX Brief follows the standard format defined in the ux-designer skill (User Profile, Trigger, Core Flow, Alternate Flows, Friction Points, Information Architecture, Design Principles, Open Questions).

Present the UX Brief to the user for review. This is a standalone review — no automatic handoff to other agents unless the user requests it.
